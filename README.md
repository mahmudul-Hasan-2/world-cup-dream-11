# 🏏 World-Cup-Dream-11

A simple React-based fantasy cricket team builder where users can select their favorite players within a budget.

### 🔗 Live Link

[ossified-cabbage.surge.sh](ossified-cabbage.surge.sh)

---

### 🚀 Key Features

- **Player Selection:** Users can add players to their squad.
- **Budget Management:** Automatic balance deduction when a player is selected.
- **Validation:** Prevents selecting more than 6 players or selecting the same player twice.
- **Responsive Design:** Fully optimized for mobile and desktop using Tailwind CSS.
- **Toast Notifications:** Real-time alerts for successful or invalid actions.

---

### 🛠️ Technologies Used

- **React.js** (Frontend Library)
- **Tailwind CSS** (Styling)
- **Lucide React** (Icons)
- **React-Toastify** (Alerts)

---

### 📦 How to Run Locally

1. Clone the repo: `git clone https://github.com/mahmudul-Hasan-2/World-Cup-Dream-11.git`
2. Install dependencies: `npm install`
3. Start the project: `npm run dev`
